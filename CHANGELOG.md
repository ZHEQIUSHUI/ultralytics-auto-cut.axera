## 使用方法

```bash
# 启动 Web UI
./start_webui.sh

# 浏览器访问
http://127.0.0.1:18080
```

### 操作流程
1. 选择模型文件 (.pt 或 .onnx)
2. 点击"预览"查看模型结构和识别结果
3. 调整参数（如需要）
4. 点击"转换"生成裁剪模型
5. 查看左右两侧的模型对比
6. 点击"下载"获取裁剪后的模型
